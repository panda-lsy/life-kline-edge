# Components (UI 组件)

此目录包含所有 React UI 组件。

## 目录结构

- `InputForm/` - 输入表单组件
- `KLineChart/` - K线图组件
- `AnalysisResult/` - 分析结果展示组件
- `ThemeSwitcher/` - 主题切换器
- `ShareCard/` - 分享卡片生成器
- `common/` - 通用组件（按钮、输入框等）

## 组件规范

- 使用函数组件 + Hooks
- 使用 TypeScript 严格类型
- 使用 Tailwind CSS 进行样式
- 组件文件名使用 PascalCase
